# usdanl-sr28-sqlite
Import the USDA National Nutrient Database (Release SR-28) into a SQLite database

Inspired & using some code from [nmaster/usdanl-sr28-mysql](https://github.com/nmaster/usdanl-sr28-mysql).

## Requirements
* Internet connection
* wget
* unzip
* dos2unix
* sed
* sqlite3

## Running
`sr28_import.sh DB_PATH`
